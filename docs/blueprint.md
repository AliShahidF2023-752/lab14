# **App Name**: Serverless Duo

## Core Features:

- Dynamic Ticket Pricing: Calculates the dynamic ticket price based on base price, demand, and days until event.
- Simple Sum Function: Calculates the sum of two provided numbers.
- Input Forms: Provides user-friendly forms to input values for both the ticket pricing and sum functions.
- API Interaction: Connects the forms to the serverless functions using API calls.
- Real-time output display: Dynamically displays the result of API calls.

## Style Guidelines:

- Primary color: Saturated purple (#9D4EDD) to suggest both creativity and reliability.
- Background color: Light, desaturated purple (#F1E4FF).
- Accent color: Indigo (#5A189A), for a harmonious and distinctive look.
- Body and headline font: 'Inter', sans-serif, for a modern, machined look.
- Simple, intuitive layout with clear sections for each function.
- Subtle animations during API calls to give users feedback during calculations.
